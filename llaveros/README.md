# Configuración de Molinete con llaveros

Esta sección reúne las guías del control de acceso por molinete con llaveros o tarjetas: la instalación en la sede del cliente y el enrolamiento de credenciales de los socios.

{% hint style="info" %}
Si el molinete es Tango, consultá la sección [Configuración Molinetes Tango](../tango/README.md): la instalación es muy similar, pero con algunas diferencias puntuales. Si el ingreso se valida por teclado (DNI) en lugar de llavero o tarjeta, consultá [Configuración Molinete Teclado](../teclado/README.md).
{% endhint %}
