# Configuración Molinetes Reloj (Huella)

Esta sección reúne las guías del control de acceso con reloj de huella digital, donde el propio reloj toma y almacena las huellas de los socios: instalación, enrolamiento, búsqueda y depuración de datos.

{% hint style="info" %}
Si la instalación usa un sensor de huella externo (por ejemplo, un ZK9500) en lugar de la huella del propio reloj, consultá la sección [Configuración Reloj 9500](../reloj-zk9500/README.md).
{% endhint %}
